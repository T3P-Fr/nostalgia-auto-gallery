# Ulteam — Manifeste

**Un dashboard que tout le monde peut piloter. Évident.**

---

## Pourquoi Ulteam existe

Les outils existants (Shopify, PrestaShop, Directus dashboards) sont conçus par des devs pour des devs.

**L'utilisateur réel ?**
- Ton beau-frère qui conduit une pelleteuse
- Le petit commerçant qui veut vendre
- La PME qui n'a pas de ressource tech

Ils abandonnent. Ils se perdent. Ils appellent quelqu'un au secours.

**Ulteam c'est :** un framework + une philosophie où chaque action est **évidente du premier coup**.

---

## Principes non-négociables

✅ **Clarté radicale** — Si tu dois lire la doc, on a échoué  
✅ **Accessible à tous** — Même à quelqu'un sans expérience tech  
✅ **Visuel avant tout** — Montre, ne raconte pas  
✅ **Pas de jargon** — "Ajouter un produit", pas "instantier une ressource"  
✅ **Peu de choix** — Mieux vaut 1 façon obvious que 5 options paralysantes  

---

## Ce qu'on ne veut PAS

❌ Concurrencer Radix/Shadcn (déjà bon pour les devs)  
❌ Être "complet" (80% des features pour 20% des gens)  
❌ Suivre les tendances design (minimalisme, glass morphism, etc.)  
❌ Ajouter de l'IA "juste parce que" (si ça rend ça flou, non)  

---

## La cible réelle

- Petits commerçants (e-commerce)
- PME avec un site simple
- Créatifs qui veulent juste vendre
- Pas les agences, pas les devs

**Le test ultime :** ton beau-frère à la pelleteuse arrive, clique 3 fois, c'est fait.

---

## L'architecture

**Slyk** = composants headless (logique + interaction)  
**Zoé** = thème CSS (clarity + beauty)  
**Ulteam** = le framework complet (Slyk + Zoé + intégrations)
