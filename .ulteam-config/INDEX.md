# Ulteam Config — Index

**Configuration centralisée du framework Ulteam.**

Sauvegarde ce dossier sur clé USB, Drive, ou synchro-le partout.

---

## 📋 Fichiers

| Fichier | Rôle | Change comment ? |
|---------|------|------------------|
| **ULTEAM.md** | Vision + manifeste | Rarement (quand la philo évolue) |
| **ROADMAP.md** | Priorités + blocs actifs | Souvent (1-2x par semaine) |
| **DECISIONS.md** | Journal des décisions | Immutable (ajout seulement, jamais modifié) |
| **CLAUDE.md** | Règles globales (tags, ODE2JS, etc.) | Rarement |
| **ODE2JS.md** | Philosophie JS + JSDoc | Quasi-immutable |

---

## 🎯 Comment utiliser

1. **Avant chaque conversation** : relis ULTEAM.md (ta vision)
2. **Pendant une conversation** : Claude utilise ROADMAP.md + DECISIONS.md
3. **Avant de merger** : valide que DECISIONS.md est à jour
